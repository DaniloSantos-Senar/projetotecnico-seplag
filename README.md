# Vue Docker App

Este projeto é uma aplicação Vue.js empacotada em um container Docker. Abaixo estão as instruções para instalação e uso.

## Pré-requisitos

- Docker
- Docker Compose

## Instalação
1. Construa a imagem Docker:
   ```
   docker-compose build
   ```

2. Inicie a aplicação:
   ```
   docker-compose up
   ```
